Award Rest API

When I first recieved this project, I was excited at the opportunity to use my knowledge of API's from school here at my internship at Workday. The premise was pretty simple. The goal of my summer project was to allow the customer to create their own tab with their outside information onto an award. In order to accomplish this, we first had to test to see if this concept could work. I went forward and used an app called Intellij which utilized VPS, which is similar to an html building tool and enables developers to display data from REST APIs into Workday and create your own page/custom task.
